Об'єкт 1: Сенсор
- sensor_id: унікальний ідентифікатор сенсора
- type: тип сенсора (температура, дим, тиск)
- location: місце встановлення
- status: активний / неактивний

Об'єкт 2: Інцидент
- incident_id: унікальний ідентифікатор інциденту
- timestamp: час виникнення
- severity: рівень критичності
- status: новий / в обробці / закритий

Об'єкт 3: Оператор
- operator_id: унікальний ідентифікатор оператора
- name: ім'я оператора
- role: роль у системі
- availability: доступність

Об'єкт 4: Ресурс
- resource_id: унікальний ідентифікатор ресурсу
- type: тип ресурсу (екіпаж, обладнання)
- location: поточне місцезнаходження
- status: вільний / задіяний
