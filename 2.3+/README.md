# Прототип інформаційно-аналітичної системи (ІАС)

## Мета роботи
Даний прототип інформаційно-аналітичної системи розроблено в межах магістерського дослідження з метою автоматизації збору, обробки та аналізу даних про інциденти, що виникають у процесі моніторингу об'єктів.

## Проблема, що вирішується
У реальних умовах управління критичними об'єктами значна кількість даних надходить з різних джерел (сенсори, оператори, ресурси), що ускладнює оперативне прийняття рішень. Відсутність єдиного інтерфейсу та автоматизованого аналізу призводить до затримок реагування.

## Опис прототипу
Прототип ІАС включає:
- локальну базу даних SQLite;
- бекенд-модуль на Python для доступу до даних;
- логічний механізм перевірки станів («WARNING»);
- веб-інтерфейс на основі Streamlit.

## Дані, що обробляються
Система працює з такими сутностями:
- сенсори;
- інциденти;
- оператори;
- ресурси реагування.

Дані зберігаються в реляційній базі даних та можуть бути використані для аналітичних запитів.

## Користь для користувача
Розроблений прототип дозволяє:
- оперативно переглядати поточний стан ресурсів;
- аналізувати інциденти в режимі реального часу;
- підтримувати прийняття рішень командиром або аналітиком;
- використовувати систему як основу для подальшого розширення.

## Висновок
Прототип підтверджує можливість побудови інформаційно-аналітичної системи з використанням сучасних інструментів Python та може бути масштабований до повнофункціонального програмного продукту.
